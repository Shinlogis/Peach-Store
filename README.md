## 🗃️ 도메인 이미지 저장 폴더

저장 경로 예시

`C:\Dev\Peach-Store-workspace\.metadata\.plugins\org.eclipse.wst.server.core\tmp0\wtpwebapps\peachStore\data\subcategory_17\저장시간.jpg`

`지정명칭_pk` 폴더에 `저장시간.jpg` 파일로 저장합니다.

**지정명칭**
- 상위카테고리: category
- 하위카테고리: subcategory
- 제품: product
- 문의: p
- 리뷰: r
